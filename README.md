# Smart-Counting-Arduino-program-using-C-
Arduino Uno countdown program using variables,functions,while loops and Serial Monitors
# Arduino Assignment-Smart Countdown

This is my submission for Assignment 1 in the programming C++ For Engineers using arduino at Ghana Communication Technology University

##What this  program does
It counts down from 5 to 1 using a while loop. On each step:
-It prints "Count N" to the serial monitor
-It blinks the on board LED N times using a function called flashLED()
When the countdown finishes,it prints"--Countdown Complete--"

##Concepts demonstrated
-The while loop
-Variables(int)
-Functions with parameters
-Digital Output
-The serial Monitor

##How to run it
1.Open the .ino file in the Arduino IDE
2.Connect an Arduino Uno via USB
3.Select tools>board.Arduino Uno and the correct port
4.Click upload, then open tools>Serial monitor(9600 baud)
